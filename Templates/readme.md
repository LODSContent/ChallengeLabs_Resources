# Using the template with the Content Assistant

- Open a lab with existing content and edit the Markdown.
- Select the Content Assistant tab on the left.
- Select Improve Content.
- Select **All Content**.
- In the textbox, enter `Convert the lab markdown to conform with the template and instructions provided in the Reference Material.`
- Choose **Claude Opus 4.6** for the model.
- Deselect the proofreading options.
- Deselect the content block options.
- In Reference Material, add the following url: `https://github.com/LODSContent/ChallengeLabs_Resources/raw/refs/heads/master/Templates/CLabs-Template-v4-with-Instructions-2025-10-14.md++`
- Select **Formatting/style guide**.
- Choose **Submit**.
- When finished processing, scan the results and accept them if it looks good.
